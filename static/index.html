EyeDee.Me
BrowserID Optimized email addresses

eyedee.me is an example "BrowserID primary" - that is, a web site that you can get a (non-functional) email address that is optimized for use with BrowserID.

This is not a real product. It is just an example to show people how they can build first-class BrowserID support into their existing product (products like gmail, that already have a login that users strongly identify with and is an email address).

To get your very own eyedee.me address so you can try it out, just sign in.
